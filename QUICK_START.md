## ğŸ’¡ Pro Tips

1. **Jekyll Watch Mode**: When running `bundle exec jekyll serve`, Jekyll watches for changes and auto-rebuilds. Just refresh your browser!

2. **Blog Post Format**: File names MUST be `YYYY-MM-DD-title.md` in the `_posts/` folder

3. **Categories**: Add 1-3 categories per post for better organization

4. **Images**: Create an `assets/img/` folder and reference images like: `![alt text](/assets/img/image.jpg)`

5. **No Need to Code**: Everything is configured. You just need to edit the content files

6. **Custom Domain**: If you buy a domain, add it in GitHub Pages settings and update `url:` in _config.yml
